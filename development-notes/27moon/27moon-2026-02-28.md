# Дата: 2026-02-28

- **Что было сделано:**

  Сделала кастомные токены для Mantine компонентов.
  Провела рефакторинг стилей для существующих страниц: заменила статические значения на стандартизированные токены из обновленной темы.

- **Проблемы и решения (или попытки):**

Поскольку каждый в команде реализует свою страничку, а детальной спецификации по отступам, марджинам и т.п. для каждого элемента нет - начали возникать расхождения в реализации.
Для того чтобы привести проект к единому виду, расширила стандартные токены Mantine,добавив единую систему отступов, радиусов, line-height и т. п., и добавила кастомные.
Теперь вместо произвольных значений используются стандартизированные (например, xl для карточек или lg для кнопок).
