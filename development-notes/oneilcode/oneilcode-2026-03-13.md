## Code Review

Разбиралась в PR (feature: stack widget), который написал @D15ND (https://github.com/RSSAgents/tandem/pull/158).

Что делает: Обучающий виджет, который через вопросы и анимации объясняет разницу между LIFO и FIFO. Основная логика в `src/components/features/widgets/Stack`
Виджет состоит из двух основных частей: вопросы и анимации. Сначала пользователь отвечает на вопрос про LIFO, затем видит анимацию стека, потом аналогично для FIFO. Управление состоянием через хук useStackAnimation, где с помощью таймеров добавляются и удаляются элементы. Логика вопросов пока задублирована для LIFO и FIFO, но в целом работает.

Структура: В папке компонента есть отдельные файлы для констант (.constants.ts), хуков (.hooks.ts) и стилей (.module.css). Сначала не сразу поняла, зачем выносить хуки и константы локально, если в проекте есть общие папки, но почитала - такая практика тоже имеет смысл для изолированной логики виджета.

Что можно улучшить: разбить на более мелкие компоненты `Stack.tsx` - сейчас почти 200 строк. Можно вынести Question, Animation и FeedbackMessage в отдельные файлы. Это сделает код читаемее и упростит тестирование (комментарий к PR оставила)
