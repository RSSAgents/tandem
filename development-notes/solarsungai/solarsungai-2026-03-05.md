Начала делать AI компонент.
- Решила создать две разные роли для ИИ - Interviewer и Teacher - первый проводит интервью, второму можно задавать рандомные вопросы.
- Решила делать Stress Mode - на ответ дается только 60 секунд. Если пользователь не успел ответить за 60 секунд, ИИ засчитывает провал по вопросу или начинает подгонять (если выбрана роль строгого учителя).
- Возможно, будет интересно добавить возможность скачать PDF файл с рекомендациями после прохождения интервью.
- Будет возможность подсказок - нажимаешь кнопку в случае если не знаешь ответ, ИИ направляет.
- Возможно, сделаю AI баттл - когда ИИ разговаривает сам с собой (из этого диалога можно выяснить что-то, чего раньше не знал и не догадывался спросить)
Пример промта: "Сейчас начнется спор. Ты - уитель, но представь, что есть еще один эксперт. Обсудите "тема" с разных точек зрения (например, Senior vs Architect). Выдай диалог на 4-5 реплик."
- Будет возможность аудио интервью - записывать речь и проигрывать ответы ИИ. Использовать Web Speech API.
- Можно добавить анимированные аватары для AI, у которого будут разные роли (строгий ментор, мягкий наставник).
- Думаю, что добавлю Code Runner чтобы можно было редактировать и запускать код. Пользователь пишет код прямо в редакторе. По кнопке Запустить код исполняется, и результат выводится рядом. ИИ может не просто обсуждать теорию, а сказать "Напиши функцию fibonacci, и я проверю, как она работает".
- При успешном прохождении можно реализовать появление конфетти (библиотека canvas-confetti).
- Skill Map (радар-диаграмма навыков). После завершения интервью ИИ может выдавать оценку не просто прошел/не прошел, а по критериям. Красивая диаграмма (Radar Chart из библиотеки Mantine Charts), которая показывает: "Синтаксис: 8/10", "Алгоритмы: 4/10", "ООП: 9/10".
Если что-то из этого не успею реализовать, можно убрать. Но думаю, что следующей недели хватит, чтобы сделать полностью.
