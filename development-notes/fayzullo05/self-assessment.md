# Два личных Feature Components

## 1. Fill Blanks Widget

Интерактивный обучающий компонент, реализованный мной.

Позволяет пользователю выбирать ответы для пропусков на основе кода и проверять их через API.
После проверки отображается результат, подсчитывается количество правильных ответов и общий score.

Основные функции:
- выбор ответов
- проверка через API
- отображение correct / incorrect
- подсчет результатов
- навигация между вопросами
- сохранение состояния ответов
- итоговый результат в модальном окне

---

## 2. Auth Flow (Login + Register)

Реализация пользовательского сценария аутентификации.

Включает страницы входа и регистрации с полной логикой работы форм.

Основные функции:
- ввод и валидация данных
- обработка ошибок
- переход после успешного входа
- регистрация с подтверждением пароля
- отображение успешной регистрации
